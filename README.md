Мой Киносалон

Персональный дневник просмотра фильмов для Android.

Описание

Приложение "Мой Киносалон" позволяет пользователям вести учет просмотренных фильмов, оценивать их, добавлять рецензии и анализировать свои кинопредпочтения.

Основные функции

Реализовано

- **CRUD операции**: Создание, чтение, редактирование и удаление фильмов
- **Поиск**: Поиск фильмов по названию, режиссеру, актерам
- **Сортировка**: По дате добавления, оценке, году выпуска, алфавиту
- **Фильтрация**: По статусу просмотра (Просмотрено, Хочу посмотреть, В процессе)
- **Статистика**: Общая статистика коллекции (количество, просмотренные, средняя оценка)
- **Экспорт данных**: Экспорт коллекции в JSON и CSV форматы
- **Темная тема**: Поддержка светлой и темной темы
- **Настройки**: Экран настроек с выбором темы и управлением данными

 Технологии

- **Kotlin** - язык программирования
- **Room** - локальная база данных
- **Material Design 3** - UI компоненты
- **Navigation Component** - навигация между экранами
- **ViewModel & LiveData** - управление состоянием
- **Coroutines & Flow** - асинхронные операции
- **ViewBinding** - привязка представлений
- **Gson** - работа с JSON

Установка и запуск

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Синхронизируйте Gradle зависимости
4. Запустите приложение на эмуляторе или устройстве

Требования

- Android 7.1 (API 25) или выше
- Android Studio Arctic Fox или новее

 Архитектура

Приложение использует архитектуру **MVVM (Model-View-ViewModel)**:

- **Model**: Room база данных с сущностями
- **View**: Fragments и Activities
- **ViewModel**: Управление состоянием и бизнес-логикой

База данных

Используется Room для локального хранения данных. База данных содержит таблицу `movies` с полями:
- id, title, year, director, actors, description
- rating, status, watchDate, review, createdAt
- posterPath (для будущей реализации)
