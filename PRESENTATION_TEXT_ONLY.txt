Идея проекта & Ключевые преимущества

ПРОБЛЕМА:
Сложность самостоятельно систематизировать просмотренные фильмы, анализировать свои предпочтения и находить интересные киноленты для будущего просмотра.

РЕШЕНИЕ "МОЙ КИНОСАЛОН":
Умный оффлайн-трекер, который помогает вести учет фильмов, анализирует ваши предпочтения и предоставляет детальную статистику.

ЧТО ВЫДЕЛЯЕТ ПРИЛОЖЕНИЕ:

✓ Персонализация под пользователя: Каждый пользователь ведет свою коллекцию с авторизацией и гостевой режим

✓ Расширенная аналитика: Алгоритм анализирует вашу коллекцию (любимые режиссеры, лучшие годы, средние оценки) и показывает детальную статистику

✓ Полная автономность: Работает без интернета. Все данные защищенно хранятся на устройстве в Room Database

✓ Умная фильтрация: Множественные фильтры по статусу, году, оценке с возможностью комбинирования

✓ Мультимедиа: Загрузка постеров из галереи или съемка с камеры для визуального оформления коллекции

✓ Напоминания: Локальные уведомления о фильмах из списка "Хочу посмотреть" и ежедневные напоминания

Демонстрация: Как это работает?

ПУТЬ ПОЛЬЗОВАТЕЛЯ ЗА 60 СЕКУНД:

• Регистрация/Вход: Создает аккаунт или входит в систему. Доступен гостевой режим для быстрого старта

• Добавляет фильмы: Вводит название, год, режиссера, актеров, описание и загружает постер

• Оценивает и классифицирует: Ставит оценку (1-5 звезд) и выбирает статус (Просмотрено / Хочу посмотреть / В процессе)

• Ищет и фильтрует: Использует поиск по названию/режиссеру/актерам и применяет фильтры по статусу, году, оценке

• Сортирует коллекцию: Выбирает сортировку по дате, оценке, году или алфавиту

• Анализирует статистику: Просматривает общую статистику (всего фильмов, просмотрено, средняя оценка) и расширенную (любимый режиссер, самый просматриваемый год, год с лучшими оценками)

• Экспортирует данные: Выгружает коллекцию в JSON или CSV для резервного копирования

• Получает напоминания: Настраивает напоминания о фильмах и ежедневные уведомления

ИТОГ: Персональный кино-дневник в вашем кармане, который помогает систематизировать и анализировать ваши кинопредпочтения

Архитектура и Технологический стек

СОВРЕМЕННЫЙ СТЕК ТЕХНОЛОГИЙ:

• Язык: Kotlin

• UI: Material Design 3, ViewBinding, Navigation Component

• Локальная БД: Room Persistence Library

• Асинхронность: Kotlin Coroutines & Flow

• Архитектура: MVVM (Model-View-ViewModel)

• Фоновые задачи: AlarmManager (для локальных напоминаний)

• Мультимедиа: Glide (загрузка изображений), FileProvider (безопасный доступ к файлам)

• Экспорт данных: Gson (JSON), CSV форматирование

ТРЕХСЛОЙНАЯ АРХИТЕКТУРА:

Presentation Layer (UI)
- Fragments (MovieListFragment, StatisticsFragment, SettingsFragment)
- Activities (LoginActivity, RegisterActivity, AddEditMovieActivity, MovieDetailActivity)
- ViewModel (MovieViewModel)
- ViewBinding для привязки представлений

Domain Layer (Логика)
- Repository (MovieRepository, MovieListRepository)
- Use Cases (фильтрация, сортировка, поиск)
- Бизнес-логика обработки данных

Data Layer (Данные)
- Room Database (MovieDatabase)
- DAO (MovieDao, UserDao, MovieListDao)
- Entities (Movie, User, MovieList, MovieListItem)
- TypeConverters для enum

Что было реализовано

ПОЛНОЦЕННОЕ PRODUCTION-ГОТОВОЕ ПРИЛОЖЕНИЕ:

• 12+ экранов: От авторизации до детальной статистики и настроек

• Полный цикл работы с фильмами: Добавление → Просмотр → Редактирование → Удаление → Анализ

• Система авторизации: Регистрация, вход, гостевой режим, восстановление пароля

• Умная фильтрация и сортировка: Множественные фильтры с комбинированием, 8 типов сортировки

• Расширенная статистика: Базовая статистика (количество, просмотрено, средняя оценка) и расширенная (любимый режиссер, самый просматриваемый год, год с лучшими оценками)

• Мультимедиа: Загрузка постеров из галереи, съемка с камеры, отображение в списке и деталях

• Система напоминаний: Локальные уведомления для отдельных фильмов и ежедневные напоминания

• Экспорт данных: Выгрузка коллекции в JSON и CSV форматы

• Темная тема: Поддержка светлой, темной и системной темы с автоматическим применением

• Пользовательские списки: Создание и управление кастомными списками фильмов

ГЛАВНОЕ ТЕХНИЧЕСКОЕ ДОСТИЖЕНИЕ:
Гармоничное сочетание красивого, отзывчивого интерфейса (Material Design 3) с сложной бизнес-логикой (умная фильтрация и статистика) и надежным локальным хранением (Room Database) с поддержкой многопользовательского режима.
